##ドット絵エディター
このゲームの最大の特徴は、キャラクターを自分で一から描ける点です。
キャンバス設定: 16x16、32x32、64x64のサイズから選択可能です

##モンスターの育成とカスタマイズ
図鑑から、仲間にしたモンスターを自由に強化できます。
ステータス配分: バトルで得たポイントを「攻撃・速度・体力・知力」に自由に割り振ります。
特性の付与: 攻撃時に「毒・マヒ・眠り・混乱」の状態異常を付与する特性を習得させることができます。
パッシブスキル: 「全体攻撃」や「自己再生」「限界突破」など、スキルを最大2つまで習得可能です。
装備システム: ダンジョンで拾った「指輪」を装備して能力を底上げできます。条件（全ステータス100以上など）を満たせば、2つ目の装備枠を解放することも可能です。

##バトルシステム
最大3体のパーティを編成し、100階層＋αのダンジョンに挑みます。
スピード感のあるバトル: 素早さに基づいたターン制バトルです。倍速モードも搭載されています。。
特殊アクション: 特定のスキルを持つモンスターが仲間を「庇う」など、パーティ編成によるシナジーが重要になります。

##ダンジョン探索
階層構造: 通常の階層に加え、10階ごとの「ボス階」が存在します。
ボスラッシュモード: クリア済みのボス階だけを選んで挑戦し、効率よく強力な指輪を集めることができます
エンドコンテンツ: 100階をクリアした猛者だけが挑戦できる「エクストラステージ」には、さらに強力な敵が待ち受けています。

##パラメーター（ステータス）
攻撃（power）: 敵に与えるダメージの基本値です。
速度（speed）: 行動順の速さに影響します。また、敵との速度差がある場合、攻撃を回避する確率（回避率）が上昇します。
体力（hp）: バトル中のHP（ヒットポイント）に影響します。パラメーター1につきHP10として換算されます。
知力（intel）: クリティカル（強撃）の発生率に直結します。
クリティカル倍率: 通常は2倍ですが、知力が100を超えると1につき0.01倍ずつ加算されます（例：知力101なら2.01倍、知力150なら2.5倍）。

##特性（追加効果）
攻撃時に一定確率で敵に付加できる特殊な状態異常です。習得には5pt必要です。
毒: 攻撃時に、与えたダメージの30%をさらに追加ダメージとして与えます。
マヒ: 30%の確率で発動し、相手を「マヒ」状態にします。マヒした敵は25%の確率で行動不能になります。
眠り: 20%の確率で発動し、相手を「眠り」状態にします。眠った敵は行動できませんが、ターン経過（30%）や攻撃を受けた際（50%）に目が覚めることがあります。
混乱: 30%の確率で発動し、相手を「混乱」状態にします。混乱した敵は、味方を攻撃してしまう可能性があります。

##パッシブ（常時発動スキル）
バトル中に自動で効果を発揮する強力な能力です。1つ目は15pt、2つ目は30ptで習得でき、最大2つまでセット可能です。
全体攻撃: 50%の確率で、敵全体を一度に攻撃します。
連続攻撃: 自分の速度が相手の1.5倍以上ある場合、2回連続で攻撃を行います。
庇う: 70%の確率で、味方が受けるダメージを代わりに肩代わりします。
自己再生: 毎ターン、自分の最大HPの10%を回復します。
状態異常無効: 敵から受ける状態異常を50%の確率で防ぎます。
限界突破: 1ターン溜める代わりに、次のターンに2.5倍の極大ダメージを叩き込みます。

##指輪（装備品）
ダンジョンのボス階層（10階ごと）で稀にドロップする貴重なアイテムです。
種類と効果: 「力の指輪」「速度の指輪」「体力の指輪」「知力の指輪」の4種類があり、拾った階層の数値分、ステータスが直接加算されます。
装備スロット: 初期状態では1つですが、以下の条件を満たして50pt消費することで、2つ目のスロットを解放できます。
解放条件: 全てのパラメーター（攻撃・速度・体力・知力）が、装備補正込みで100以上であること。
